🤖 AI Game Control — Help

What is the Lap ID?

A Lap is a single AI vs AI chess game stored on the server. Each lap has a unique numeric or string identifier called the Lap ID. You need to enter this ID to connect the board to the corresponding game stream.

You can find existing Lap IDs in the Replay tab.

First, you need to replay the chosen lap, once all events are replayed, you can start the AI game and watch its continuation live.

Start AI

Entering a non existing valid Lap ID (any string) and clicking Start AI will open a live Server-Sent Events (SSE) stream and replay the moves on the board in real time as the AI players make them.

Stop AI / Stop Replay

Click Stop AI (or Stop Replay during a replay) at any time to disconnect from the stream and freeze the board at the current position.

Replay tab

The Replay tab lists all previously recorded games. Click Replay next to any game to watch it move-by-move on the board without needing to enter a Lap ID manually.
