Algebraic Notation Guide

Chess notation uses letters and numbers to describe moves:

  • Files (columns): a-h (left to right)
  • Ranks (rows): 1-8 (bottom to top for White)
  • Pieces: K=King, Q=Queen, R=Rook, B=Bishop, N=Knight (Pawns have no letter)

Special Symbols

  • x - Capture (e.g., Nxe5)
  • + - Check (optional, auto-detected)
  • # - Checkmate (optional, auto-detected)
  • = - Promotion (e.g., e8=Q)

Move Notation Examples

e4 - Move pawn to e4, Nf3 - Move knight to f3, Bc4 - Move bishop to c4, Qd5 - Move queen to d5, Rad1 - Move rook from a-file to d1 (when disambiguation needed), exd5 - Pawn on e-file captures on d5, Nxe5 - Knight captures on e5, O-O - Castle kingside (short castle), O-O-O - Castle queenside (long castle), e8=Q - Pawn promotion to queen
